Why SQLite?

Because it is simple and does not need a separate database server.

cursor?

A cursor is used to execute SQL commands.
Think of cursor as a worker that talks to the database.

1. What is FastAPI?

Answer:
FastAPI is a modern Python web framework used to build APIs quickly.
It is fast, easy to use, and automatically creates API documentation.
app = FastAPI() -> This creates a FastAPI application object.
